if 语句

if 语句简介

一个 if 语句 由一个布尔表达式后跟一个或多个语句组成。

如果布尔表达式为 true，则 if 语句内的代码块将被执行。如果布尔表达式为 false，则 if 语句结束后的第一组代码将被执行。

示例：if 语句


#include <stdio.h>

int main ()
{
    int a = 10;

    if( a < 20 )
    {
        printf("a 小于 20\n");
    }
    printf("a 的值是 %d\n", a);

    return 0;
}

上面的代码会检查变量 a 的值，如果 a 小于 20，输出 "a 小于 20"。
